Для дашборда использовался набор данных Amazon Sales Dataset за 2022-2023 года

В нём собраны данные о продажах на Amazon — заказы, товары и показатели по ним. 
На основе этих данных можно смотреть, как менялись продажи со временем, какие категории товаров продавались лучше, 
и строить другие графики и отчёты по продажам.
