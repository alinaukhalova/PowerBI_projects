В этом репозитории собраны проекты в области продукт-аналитики и визуализации данных, 
реализованные с помощью Power BI. Здесь представлены дашборды, построенные на реальных 
и открытых наборах данных, с акцентом на анализ продаж, поведения покупателей и ключевых 
бизнес-показателей.

Каждый проект включает:
- файл дашборда (.pbix);
- скриншоты визуализаций;

## Стек инструментов
- Power BI Desktop
- Power Query (M) 
- DAX
